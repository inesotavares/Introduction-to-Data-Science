The Euclidean distance between two points in either the plane or 3-dimensional space measures the length of a segment connecting the two points.¶

Minkowski Distance¶

Minkowski distance is a generalization of the Euclidean distance and the Manhattan distance. Minkowski distance is applied in machine learning to find out distance similarity. If p=1 it is Manhattan distance, and if p=2 it is Euclidean distance.¶

In [66]:
def Mink_D(x,y,p):
    n=len(x)
    if len(x)!=len(y):
        return 'Erro'
    else:
        return round((sum((abs(c1-c2))**p))**(1/p),5)

Mahalanobis Distance¶

Mahalanobis distance between two vectors, x and y, where S is the co-variance matrix. Co-variance of two feature indicated how values of two features are varying together. It measure how values of one feature are varying according to values of another feature.¶
In [123]:
import numpy as np
from numpy.linalg import inv
from scipy.spatial import distance

def Maha_D(M,S):
    S_Inv = np.linalg.inv(S)
    return round(distance.mahalanobis(M.iloc[0,:],M.iloc[1,:],S_Inv),5)

Minkowski distance is a flexible metric that allows you to control the sensitivity to different dimensions using the parameter. When p=2, it corresponds to the the Euclidean distance¶

For uncorrelated variables, the Euclidean distance equals the Mahalanobis Distance. However, if two or more variables are correlated, the axes are no longer at right angles, and the measurements become impossible with a ruler. In addition, if there is more than three variables, it's not possible to plot them in regular 3D space at all. The Mahalanobis Distance solves this measurement problem, as it measures distances between points, even correlated points for multiple variables.¶

import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D
import seaborn as sns

# Assuming you have loaded the Iris dataset as 'iris' here

fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

# Define a color map for each species
species_colors = {
    'setosa': 'red',
    'versicolor': 'green',
    'virginica': 'blue'
}

# Iterate through the data and plot points with species-specific colors
for species in iris['species'].unique():
    species_df = iris[iris['species'] == species]
    ax.scatter(species_df['sepal_length'],
               species_df['sepal_width'],
               species_df['petal_length'],
               color=species_colors[species],  # Assign color based on species
               label=species,
               edgecolor='k',
               s=60)

def markerSize(x):
    # Define upperBoundary and lowerBoundary based on your data
    upperBoundary = 3  # Example value, adjust as needed
    lowerBoundary = 1  # Example value, adjust as needed
    if (x > upperBoundary):
        return 75
    if (x < lowerBoundary):
        return 5
    else:
        return 25

markers = list(map(markerSize, iris['petal_width']))
xs = iris['petal_length']
ys = iris['sepal_width']
zs = iris['sepal_length']
ax.scatter(xs, ys, zs, s=markers, marker='o', c=iris['species'].map(species_colors))
ax.set_xlabel('Sepal Length (cm)')
ax.set_ylabel('Sepal Width (cm)')
ax.set_zlabel('Petal Length (cm)')
ax.set_title('3D Scatter Plot of Iris Dataset')
ax.view_init(20, -70)
ax.legend()

plt.show()
